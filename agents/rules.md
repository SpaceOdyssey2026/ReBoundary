# Project Rules

## Documentation Rules

1. **Synchronized Documentation Updates**:
   - Every time the codebase is changed, refactored, or extended, the contents under the `docs` folder must be updated accordingly to reflect the modifications and ensure consistency with the system's behavior.

2. **Bilingual Support in `docs`**:
   - All official documentation inside the `docs` folder must support **both Chinese and English** natively. When editing or creating new documents in `docs`, always provide the content in both languages.
